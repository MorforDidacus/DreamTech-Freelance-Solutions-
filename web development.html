HTML Tutorial Series

Learn HTML from scratch with hands-on examples and explanations

Tutorial 1: Introduction to HTML

Learn what HTML is and why it is the backbone of any web page. HTML provides the structure for web content.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Introduction to HTML</title>
  </head>
  <body>
    <h1>Welcome to HTML!</h1>
    <p>HTML stands for HyperText Markup Language.</p>
  </body>
</html>

Tutorial 2: Basic Structure of an HTML Document

This tutorial explains the fundamental structure required for any HTML document, including the DOCTYPE, html, head, and body tags.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Basic HTML Structure</title>
  </head>
  <body>
    <header><h1>Header Section</h1></header>
    <main><p>Main content goes here.</p></main>
    <footer><p>Footer information.</p></footer>
  </body>
</html>

Tutorial 4: HTML Links and Anchors

Tutorial 8: HTML Forms

Forms collect user input. In this tutorial, we build a simple form with text fields and a submit button.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Simple Form</title>
  </head>
  <body>
    <form action="/submit" method="post">
      <label for="username">Username:</label>
      <input type="text" id="username" name="username" required>
      <button type="submit">Submit</button>
    </form>
  </body>
</html>

Tutorial 24: HTML Canvas Basics

The <canvas> element lets you draw graphics on the fly using JavaScript.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Canvas Example</title>
  </head>
  <body>
    <canvas id="myCanvas" width="300" height="150" style="border:1px solid #000;">
      Your browser does not support the canvas element.
    </canvas>
    <script>
      const canvas = document.getElementById('myCanvas');
      const ctx = canvas.getContext('2d');
      ctx.fillStyle = 'green';
      ctx.fillRect(10, 10, 100, 50);
    </script>
  </body>
</html>

Tutorial 26: HTML Microdata & Schema

Add semantic meaning to your pages by using microdata attributes. This helps search engines better understand your content.

<!DOCTYPE html>
<html lang="en" itemscope itemtype="http://schema.org/Article">
  <head>
    <meta charset="UTF-8">
    <title>Microdata Example</title>
  </head>
  <body>
    <h1 itemprop="headline">Article Title</h1>
    <p itemprop="articleBody">This is the article content.</p>
  </body>
</html>

Tutorial 30: HTML Custom Elements (Web Components)

Create your own custom HTML elements using the Web Components standard.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Custom Elements</title>
  </head>
  <body>
    <!-- Define a custom element -->
    <script>
      class MyElement extends HTMLElement {
        constructor() {
          super();
          this.innerHTML = "<p>This is a custom element.</p>";
        }
      }
      customElements.define('my-element', MyElement);
    </script>
    <my-element></my-element>
  </body>
</html>
